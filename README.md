# ε-δ Explorer

미적분학의 연속성과 균등연속성을 시각화하는 도구입니다.

## 접속

- **자동 감지**: https://letsgokdh.github.io/epsilon_delta_visualizer/
- **PC 접속**: https://letsgokdh.github.io/epsilon_delta_visualizer/desktop.html
- **모바일 접속**: https://letsgokdh.github.io/epsilon_delta_visualizer/mobile.html

## 사용법

### 기본 조작

- **함수 입력**: `f(x) =` 옆 입력창에 함수 입력 (예: `1/x`, `sin(x)`, `x^2`)
- **x₀ 슬라이더**: 기준점을 이동하면 그래프에 보라색 점이 움직입니다
- **ε 슬라이더**: 허용 오차를 조절하면 파란색 ε 범위가 변합니다

### 그래프 시각화 요소

- **파란색 수평 범위**: ε 범위 (y 방향, f(x₀) ± ε)
- **주황색 수직 범위**: δ(x₀) 범위 (현재 x₀에서 계산된 델타)
  - x₀를 움직이면 이 범위도 변합니다
- **초록색 수직 범위**: 전역 δ_min (균등연속일 때 표시)
  - 모든 x₀에서 작동하는 최소 δ
  - x₀를 움직여도 고정되어 있음
  - 균등연속의 핵심: 하나의 δ가 전 구간에서 작동

### 정의역 설정

- **구간 [a, b]**: x min, x max로 범위 지정
  - 대괄호 클릭: 열린/닫힌 구간 전환
  - −∞, +∞ 체크박스: 반무한 구간 설정 (예: [0, +∞), (−∞, 5])
- **ℝ 전체**: 실수 전체. view min/max로 보기 범위만 조정
- **y 범위 자동 조정**: 함수가 잘 보이도록 y 축 자동 설정

### 프리셋 함수

| 함수 | 특징 |
|------|------|
| 1/x | x=0에서 불연속 |
| x² | 연속이지만 ℝ에서 비균등연속 (δ가 x₀에 의존) |
| sin(x) | ℝ에서 균등연속 (초록색 δ_min 고정) |
| sqrt(x) | [0,∞)에서 균등연속 |
| tan(x) | π/2에서 점근선 |
| floor(x) | 정수에서 점프 불연속 |
| sin(1/x) | x=0 근처에서 진동 |

## 균등연속성 이해하기

### 연속 vs 균등연속

**연속 (Continuity)**
- 각 점 x₀마다 δ를 찾을 수 있음
- δ는 x₀와 ε 둘 다에 의존: δ(x₀, ε)
- x₀를 움직이면 주황색 δ 범위가 변함

**균등연속 (Uniform Continuity)**
- 모든 점에서 작동하는 하나의 δ가 존재
- δ는 ε에만 의존: δ(ε)
- 초록색 δ_min이 고정되어 있음

### 시각적 판별법

1. **x₀ 슬라이더를 움직여보세요**
2. **주황색 δ(x₀) 범위를 관찰**:
   - 계속 변하면 → 비균등연속
   - 항상 초록색 범위 안에 있으면 → 균등연속

예시:
- `x^2`를 ℝ에서: x₀ 커질수록 주황색 범위가 작아짐 → 비균등연속
- `sin(x)`를 ℝ에서: 주황색 범위가 초록색 안에서만 움직임 → 균등연속

## 판정 결과

- **연속성**: 불연속점, 점근선, 미정의점 자동 감지
- **균등연속성**:
  - 전체 구간에서 δ_min 계산
  - δ_min > 0이고 충분히 크면 균등연속
  - 닫힌 유계 구간의 연속함수는 자동으로 균등연속 (Heine-Cantor 정리)
