# ε-δ Explorer

미적분학의 연속성과 균등연속성을 시각화하는 도구입니다.

## 사용법

### 기본 조작

- **함수 입력**: `f(x) =` 옆 입력창에 함수 입력 (예: `1/x`, `sin(x)`, `x^2`)
- **x₀ 슬라이더**: 기준점을 이동하면 그래프에 보라색 점이 움직입니다
- **ε 슬라이더**: 허용 오차를 조절하면 파란색 ε 범위가 변합니다
- **δ 범위**: 주황색으로 표시되며 자동으로 계산됩니다

### 정의역 설정

- **구간 [a, b]**: x min, x max로 범위 지정. 대괄호 클릭하면 열린/닫힌 구간 전환
- **ℝ 전체**: 실수 전체. view min/max로 보기 범위만 조정

### 프리셋 함수

| 함수 | 특징 |
|------|------|
| 1/x | x=0에서 불연속 |
| x² | 연속이지만 ℝ에서 비균등연속 |
| sin(x) | ℝ에서 균등연속 |
| sqrt(x) | [0,∞)에서 균등연속 |
| tan(x) | π/2에서 점근선 |
| floor(x) | 정수에서 점프 불연속 |
| sin(1/x) | x=0 근처에서 진동 |

## 판정 결과

- **연속성**: 불연속점, 점근선, 미정의점 감지
- **균등연속성**: 전체 구간에서 δ_min 계산
  - 닫힌 유계 구간의 연속함수는 자동으로 균등연속 (Heine-Cantor 정리)
